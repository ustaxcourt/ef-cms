# Running this project locally

### UI (Terminal A)

##### Setup

- `npm i`

##### Running

- `npm start`

### API (Terminal B)

##### Setup

- `npm i`
- `npm run install:dynamodb`

##### Running

- `npm start`

# Electronic Filing / Case Management System
